Это самая простая и правильная инструкция для твоего проекта.

## Рекомендуемый способ (самый удобный)

### Шаг 1: Подготовка

1. Открой терминал в папке проекта:
   ```bash
   cd "/Users/konsung/Desktop/Мои_проекты/konsung-rus"
   ```

2. Добавь все изменения и сделай коммит:
   ```bash
   git add .
   git commit -m "Подготовка к первому деплою"
   ```

3. Если у тебя ещё нет репозитория на GitHub:
   - Зайди на https://github.com
   - Создай новый репозиторий (назови его `konsung-rus`)
   - **Не** ставь галочку "Add a README file"
   - Скопируй ссылку на репозиторий (HTTPS)

4. Подключи репозиторий (замени ссылку на свою):
   ```bash
   git remote add origin https://github.com/ТВОЙ_ЛОГИН/konsung-rus.git
   git branch -M main
   git push -u origin main
   ```

### Шаг 2: Деплой на Vercel

1. Зайди на сайт: https://vercel.com
2. Войди через GitHub (кнопка "Continue with GitHub")
3. Нажми большую кнопку **"Add New Project"**
4. Найди в списке репозиторий `konsung-rus` и нажми **"Import"**
5. Vercel автоматически всё настроит (это Next.js проект)

### Шаг 3: Добавь переменные окружения (ОБЯЗАТЕЛЬНО!)

Пока Vercel деплоит проект, сделай следующее:

1. После того как проект создался, зайди в него.
2. Перейди во вкладку **Settings** → **Environment Variables**
3. Добавь переменную:

   **Name:** `RESEND_API_KEY`  
   **Value:** `re_xxxxxxxxxxxxxxxxxxxxxxxx` ← твой ключ от Resend

4. Выбери окружения: **Production**, **Preview**, **Development**
5. Нажми **Save**

### Шаг 4: Передеплой

После добавления переменной нажми **"Redeploy"** на странице проекта.

---

## После деплоя

- Твой сайт будет доступен по адресу вида:  
  `https://konsung-rus.vercel.app`

- Чтобы подключить свой домен (например `konsungrus.ru`):
  1. Зайди в проект на Vercel
  2. Перейди в **Settings → Domains**
  3. Введи свой домен и следуй инструкциям

---

## Важно

- Форма обратной связи будет работать только после того, как ты добавишь `RESEND_API_KEY` в Vercel.
- Если хочешь, чтобы письма приходили с адреса `info@konsungrus.ru`, нужно будет позже настроить домен в Resend.
