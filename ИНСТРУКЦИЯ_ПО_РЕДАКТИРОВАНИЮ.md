# Инструкция по редактированию сайта Консунг Рус

Этот файл поможет тебе самостоятельно менять контент сайта без помощи программиста.

## 1. Как запустить сайт локально (чтобы видеть изменения)

1. Открой терминал в VS Code (`Terminal → New Terminal`)
2. Вставь команду и нажми Enter:

```bash
npm run dev
```

3. Дождись, пока появится строчка:
   ```
   Local: http://localhost:3000
   ```

4. Открой в браузере: **http://localhost:3000**

Теперь все изменения, которые ты сохранишь, будут сразу отображаться на сайте.

---

## 2. Как редактировать тексты на сайте

Большинство текстов сайта находится в одном файле:

**`content/site-texts.ts`**

### Примеры, что можно менять:

- Заголовок на главной странице (Hero)
- Текст в разделе «О компании»
- Заголовки и описания в разделе «Для дистрибьюторов»
- Контакты и реквизиты

### Как изменить текст:

1. Открой файл `content/site-texts.ts`
2. Найди нужный раздел (например `hero`, `about`, `distributors`)
3. Поменяй текст внутри кавычек
4. Сохрани файл (`Ctrl + S`)
5. Обнови страницу в браузере

**Пример:**
```ts
hero: {
  titleLine1: "Глюкометры и",
  titleLine2: "тест-полоски Sejoy для точного контроля",
}
```

## 3. Как добавить новый продукт (самое важное)

### Шаг за шагом:

1. **Открой файл** `content/products.ts`

2. **Добавь новый продукт** в конец массива `products` (перед последней закрывающей скобкой `]`).

   Скопируй пример из конца файла и измени данные.

3. **Добавь фото продукта**:
   - Положи изображение в папку: `public/images/products/`
   - В файле укажи путь, например: `"products/noviy-glyukometr.jpg"`

4. **Добавь инструкцию (PDF)** (по желанию):
   - Положи файл в папку: `public/pdfs/`
   - В объекте продукта укажи: `pdf: "nazvanie-instrukcii.pdf"`

5. **Сохрани файл** и обнови страницу.

### Обязательные поля при добавлении продукта:

```ts
{
  id: "unikalnyy-id",                    // Уникальное имя на английском (без пробелов)
  name: "Название продукта",             // Например: Sejoy BG-750
  category: "glucometer",                // "glucometer" или "strips"
  image: "products/nazvanie-foto.jpg",   // Путь к фото
  description: "Описание продукта...",
  specs: [                               // Список характеристик
    { label: "Время измерения", value: "5 секунд" },
    // добавляй сколько нужно
  ],
}
```

**Рекомендации:**
- Используй 8–12 характеристик в `specs`, чтобы выглядело солидно
- Добавляй `badge`, если нужно (например `"Новинка"` или `"С Bluetooth"`)
- Поле `shortFeatures` — это 2–4 коротких преимущества под названием

## 4. Как добавить фотографии и документы

### Фотографии:
- Основные фото (герой, логотипы и т.д.): `public/images/`
- Фото продуктов: `public/images/products/`

### Документы (PDF):
- Все инструкции и документы: `public/pdfs/`

После добавления файла просто укажи правильный путь в `content/products.ts` или в текстах.

## 5. Как поменять цвета и стили

**Файл:** `app/globals.css`

Там в самом верху есть переменные:

```css
:root {
  --primary: #0F4C81;        /* Основной синий */
  --accent: #E85A8C;         /* Розовый акцент */
  ...
}
```

Меняй эти цвета — они применятся по всему сайту.

**Внимание:** Если не уверен — лучше сначала спроси, прежде чем сильно менять стили.
