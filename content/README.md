# Как редактировать контент сайта

**Основная подробная инструкция** находится в файле:

**`ИНСТРУКЦИЯ_ПО_РЕДАКТИРОВАНИЮ.md`** (в корне проекта)

Открой именно этот файл — там всё разложено по шагам:
- Как запускать сайт
- Как добавлять новые продукты
- Как менять любые тексты
- Как добавлять фото и документы
- Полезные советы

---

### Кратко про файлы в этой папке:

- `products.ts` → все товары (глюкометры, полоски и будущие продукты)
- `site-texts.ts` → основные тексты сайта (заголовки, описания и т.д.)
- `index.ts` → технический файл (не трогай)

**Главный файл для тебя** — `ИНСТРУКЦИЯ_ПО_РЕДАКТИРОВАНИЮ.md` в корне проекта.
