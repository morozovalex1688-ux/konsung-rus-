# Шпаргалка по редактированию сайта

## 1. Запуск сайта (чтобы видеть изменения)

```bash
npm run dev
```

После запуска открывай: **http://localhost:3000**

---

## 2. Основные файлы для редактирования

| Задача                        | Файл                              |
|------------------------------|-----------------------------------|
| Добавить/поменять продукт    | `content/products.ts`             |
| Поменять тексты на сайте     | `content/site-texts.ts`           |
| Добавить фото товара         | `public/images/products/`         |
| Добавить документ (PDF)      | `public/pdfs/`                    |
| Поменять цвета               | `app/globals.css` (переменные)    |

---

## 3. Быстрые действия

### Поменять любой текст
1. Открой `content/site-texts.ts`
2. Найди нужный раздел (`hero`, `about`, `distributors` и т.д.)
3. Поменяй текст внутри кавычек
4. Сохрани (`Ctrl + S`)

### Добавить новый продукт
1. Открой `content/products.ts`
2. В конце массива `products` добавь новый объект (скопируй существующий)
3. Заполни поля:
   - `name`, `description`, `specs`
   - `image` — путь к фото из папки `public/images/products/`
   - `pdf` — имя файла из папки `public/pdfs/` (если есть)
4. Сохрани файл

### Добавить фото или PDF
- Фото → папка `public/images/products/`
- Инструкция → папка `public/pdfs/`
- После добавления файла укажи путь в `products.ts`

---

## 4. Полезные команды

| Действие                    | Команда                  |
|----------------------------|--------------------------|
| Запустить сайт             | `npm run dev`            |
| Остановить сайт            | `Ctrl + C` в терминале   |
| Переустановить зависимости | `npm install`            |

---

## 5. Важные правила

- **Всегда сохраняй файлы** после правок
- Работай только внутри папки `content/` (самое безопасное)
- Если что-то сломалось — напиши мне, не паникуй
- При добавлении нового продукта старайся сохранять похожий стиль описания и количество характеристик
