# Настройка отправки заявок на почту

Форма обратной связи отправляет все сообщения на `info@konsungrus.ru`.

## Рекомендуемая настройка (для надёжной доставки)

### 1. Получи API ключ Resend

1. Перейди на https://resend.com
2. Зарегистрируйся
3. Скопируй API ключ (начинается с `re_`)

### 2. Создай файл `.env.local` в корне проекта

Добавь туда следующие строки:

```env
# API ключ от Resend
RESEND_API_KEY=re_твой_ключ_сюда

# Email, на который будут приходить заявки (по умолчанию info@konsungrus.ru)
CONTACT_EMAIL=info@konsungrus.ru
```

### 3. Подтверди домен в Resend (очень важно!)

Сейчас письма отправляются с адреса `info@konsungrus.ru`.

Чтобы письма **надёжно доходили** и не попадали в спам:

1. В Resend перейди в **Domains → Add Domain**
2. Добавь домен `konsungrus.ru`
3. Добавь DNS-записи, которые покажет Resend (обычно 2 TXT + 1 CNAME)
4. Дождись подтверждения домена (может занять от 5 минут до нескольких часов)

После подтверждения домена письма будут отправляться легитимно от имени твоей компании.

---

Готово! Все заявки с формы будут приходить на `Info@konsungrus.ru`.

Если письма не приходят:
- Проверь папку «Спам»
- Убедись, что `RESEND_API_KEY` и `CONTACT_EMAIL` заданы в `.env.local` (или в переменных окружения на Vercel)
- Посмотри логи в терминале / Vercel Logs

После настройки домена в Resend можно дополнительно настроить красивый отправитель (если нужно).
