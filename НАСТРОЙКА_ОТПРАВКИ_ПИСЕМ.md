# Настройка отправки заявок на почту

Сейчас форма обратной связи полностью рабочая и отправляет письма на `info@konsungrus.ru`.

## Что нужно сделать (один раз):

### 1. Зарегистрируйся в Resend (бесплатно)

1. Перейди на https://resend.com
2. Зарегистрируйся через Google или email
3. После регистрации тебе сразу дадут API ключ (начинается с `re_`)

### 2. Добавь ключ в проект

1. В корне проекта создай файл `.env.local` (если его ещё нет)
2. Вставь туда строку:

```env
RESEND_API_KEY=re_твой_ключ_сюда
```

3. Сохрани файл и **перезапусти** dev-сервер (`npm run dev`)

### 3. (Опционально) Настрой свой домен

Сейчас письма приходят с адреса `onboarding@resend.dev`.

Чтобы письма приходили с `info@konsungrus.ru`:

1. В личном кабинете Resend → Domains → Add Domain
2. Добавь `konsungrus.ru`
3. Добавь TXT-записи в DNS своего домена (подскажут в Resend)
4. После подтверждения домена поменяй в файле `app/api/contact/route.ts` строку:

```ts
from: 'Сайт Консунг Рус <info@konsungrus.ru>',
```

---

Готово! Все заявки с формы будут приходить на `info@konsungrus.ru` в красивом оформленном виде.

Если что-то не работает — проверь консоль браузера и терминал с `npm run dev`.
