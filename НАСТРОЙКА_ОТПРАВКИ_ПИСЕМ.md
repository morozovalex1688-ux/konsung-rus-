# Настройка отправки заявок с формы на почту

Все сообщения из формы обратной связи отправляются на **Info@konsungrus.ru**.

## Пошаговая инструкция

### Шаг 1. Получи API-ключ от Resend

1. Перейди на [https://resend.com](https://resend.com)
2. Зарегистрируйся (удобно через Google)
3. После входа сразу скопируй свой API-ключ (он начинается с `re_`)

> **Важно:** Ключ показывается только один раз. Сохрани его в надёжное место.

### Шаг 2. Добавь переменные окружения

#### Вариант А — Локальная разработка

В корне проекта создай файл `.env.local` (если его ещё нет) и добавь туда:

```env
RESEND_API_KEY=re_твой_ключ_сюда

# Почта, на которую будут приходить заявки
CONTACT_EMAIL=info@konsungrus.ru

# Отправитель письма (рекомендуется после подтверждения домена)
FROM_EMAIL=Сайт Консунг Рус <info@konsungrus.ru>
```

После создания файла перезапусти dev-сервер (`npm run dev`).

#### Вариант Б — На Vercel (обязательно сделать!)

1. Зайди в свой проект на [Vercel](https://vercel.com)
2. Перейди в **Settings → Environment Variables**
3. Добавь следующие переменные:

| Название            | Значение                              | Применить к          |
|---------------------|---------------------------------------|----------------------|
| `RESEND_API_KEY`    | `re_твой_ключ_от_resend`              | Production + Preview |
| `CONTACT_EMAIL`     | `info@konsungrus.ru`                  | Production + Preview |
| `FROM_EMAIL`        | `Сайт Консунг Рус <info@konsungrus.ru>` | Production + Preview |

4. После добавления всех переменных нажми **Redeploy** проекта.

### Шаг 3. Подтверди домен в Resend (самое важное для доставки)

Если домен не подтверждён, письма могут приходить с адреса `onboarding@resend.dev` и часто попадают в спам.

Чтобы письма приходили **от имени info@konsungrus.ru** и хорошо доходили:

1. В личном кабинете Resend перейди в раздел **Domains**
2. Нажми **Add Domain**
3. Введи `konsungrus.ru` и нажми Add
4. Resend покажет DNS-записи (обычно 2 TXT-записи и 1 CNAME)
5. Добавь эти записи в DNS твоего домена (у регистратора)
6. Подожди подтверждения (от 5 минут до 2–3 часов)

После успешной верификации домена письма будут отправляться легитимно от `info@konsungrus.ru`.

## Проверка работы

1. Заполни форму на сайте и отправь тестовую заявку.
2. Проверь почту `info@konsungrus.ru` (включая папку Спам).
3. Если письмо не пришло — посмотри логи:
   - При локальной разработке — в терминале
   - На Vercel — в разделе **Deployments → View Logs**

## Частые проблемы и решения

| Проблема                        | Решение |
|--------------------------------|--------|
| Письма не приходят             | Проверь, что `RESEND_API_KEY` и `CONTACT_EMAIL` заданы в Vercel |
| Письма приходят с `onboarding@resend.dev` | Подтверди домен в Resend (Шаг 3) |
| Письма попадают в спам         | Подтверди домен + используй `FROM_EMAIL=info@konsungrus.ru` |
| Ошибка "Resend error"          | Проверь правильность ключа и что домен подтверждён |
